Installation
------------

^^^
Pip
^^^

.. code-block:: bash

    python -m pip install ligotimegps

Supported python versions: 2.7, 3.4+.

^^^^^
Conda
^^^^^

.. code-block:: bash

    conda install -c conda-forge ligotimegps

Supported python versions: 2.7, 3.5+.
